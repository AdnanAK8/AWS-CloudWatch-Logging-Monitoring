# Custom CloudWatch Dashboard

## Objective

Create a centralized dashboard for monitoring EC2 instance performance.

---

## Dashboard Name

EC2-Monitoring-Dashboard

---

## Metrics Included

| Metric | Description |
|----------|------------|
| CPU Utilization | Processor usage |
| Network In | Incoming traffic |
| Network Out | Outgoing traffic |
| Disk Read Ops | Disk read operations |
| Disk Write Ops | Disk write operations |
| Status Check Failed | Instance health |

---

## Dashboard Layout

CPU Utilization Graph

Network Traffic Graph

Disk Usage Graph

Instance Health Status

---

## Monitoring Benefits

- Performance tracking
- Resource optimization
- Capacity planning
- Quick issue detection
