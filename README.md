# AWS CloudWatch Logging and Monitoring

## Project Overview

This project demonstrates the configuration of Amazon CloudWatch Logs and the creation of a custom CloudWatch Dashboard for monitoring EC2 instances.

---

## Objectives

- Configure CloudWatch Logs.
- Monitor EC2 performance.
- Create a centralized dashboard.
- Improve visibility into cloud resources.

---

## Deliverables

### CloudWatch Logs Configured

- Centralized log collection
- Log retention management
- Real-time monitoring

### Custom Dashboard Created

- CPU Monitoring
- Network Monitoring
- Disk Monitoring
- Health Monitoring

---

## Architecture

Internet

    ▼

    ▼

    ▼

    ▼
