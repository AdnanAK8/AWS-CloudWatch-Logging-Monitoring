# CloudWatch Logs Configuration

## Objective

Configure Amazon CloudWatch Logs to collect and monitor logs generated by EC2 instances.

---

## EC2 Instance Details

- Instance Type: t2.micro
- Operating System: Amazon Linux 2023
- Monitoring Service: Amazon CloudWatch

---

## CloudWatch Agent Installation

The CloudWatch Agent is installed on EC2 instances to collect system and application logs.

### Logs Monitored

- System Logs
- Application Logs
- Apache/Nginx Logs
- Authentication Logs

---

## Log Group Configuration

| Parameter | Value |
|------------|---------|
| Log Group Name | EC2-System-Logs |
| Retention Period | 30 Days |
| Monitoring Region | us-east-1 |

---

## Benefits

- Centralized log management
- Real-time monitoring
- Troubleshooting support
- Improved visibility
